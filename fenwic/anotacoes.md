## Diferença entre C e C++

C é uma linguagem de programação procedural, enquanto C++ é uma linguagem multiparadigma que suporta tanto programação procedural quanto programação orientada a objetos. Algumas diferenças principais incluem:

- **Paradigma**: C é procedural, enquanto C++ suporta orientação a objetos.
- **Encapsulamento**: C++ permite encapsular dados e funções em classes, enquanto C não possui esse recurso.
- **Herança e Polimorfismo**: C++ suporta herança e polimorfismo, características ausentes em C.
- **Bibliotecas**: C++ possui a STL (Standard Template Library), que oferece estruturas de dados e algoritmos prontos.
- **Funções**: Em C++, funções podem ser sobrecarregadas, enquanto em C isso não é possível.
- **Segurança de Tipos**: C++ é mais rigoroso com segurança de tipos em comparação com C.

Ambas as linguagens têm suas vantagens e são amplamente utilizadas em diferentes contextos.
